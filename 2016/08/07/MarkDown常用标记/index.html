MarkDown常用标记

发表于 2016-08-07

MarkDown是一种标记语言，他的语法全由一些符号所组成，这些符号经过精挑细选，其作用一目了然，这里记录最近学习的一些MarkDown常用的标记。
无序列表
Mardown写法:

* 第一行
* 第二行
* 第三行

+ 第一行
+ 第二行

- 第一行
- 第二行

1. 第一项
2. 第二项
3. 第三项

显示效果:

  • 第一行
  • 第二行
  • 第三行
  • 第一行
  • 第二行
  • 第一行
  • 第二行
  1. 第一项
  2. 第二项

  3. 第三项

  4. 第一行代码

Mardown写法:
private void getSum(int a,int b){
return a+b;
}
显示效果:

private void getSum(int a,int b){
    return a+b;
}

2. 分割线

Mardown写法:

* * *
***
---

显示效果:

*

3. 强调

Mardown写法:

*这是一段强调的文字*<br>
_这是一段强调的文字_<br>
**这是一段被强调的文字**<br>
这是一段被__强调__的文字<br>
这是一段被\*强调\*的文字<br>

显示效果:
这是一段强调的文字

这是一段强调的文字

这是一段被强调的文字

这是一段被强调的文字

这是一段被*强调*的文字

4. 一小段行内代码

Mardown写法:
User the private String getName() function

显示效果:
User the private String getName() function

5. 网址链接

Mardown写法:
我的github地址：https://github.com/Aicibg\
我的邮箱地址：dhao729176112@163.com\
显示效果:
我的github地址：https://github.com/Aicibg

我的邮箱地址：dhao729176112@163.com
